# Biographical Index

Quick reference for every major figure profiled across the seven era files, ordered by birth year.

| Years | Figure | Known for | Era |
|---|---|---|---|
| 1791–1871 | Charles Babbage | Designed the Analytical Engine — first programmable computer concept. Separated memory from processor. | [I](era-1-mathematical-foundations.md) |
| 1815–1852 | Ada Lovelace | First computer programmer. Note G (1843) — Bernoulli numbers algorithm. Theorized general-purpose symbolic computation. | [I](era-1-mathematical-foundations.md) |
| 1815–1864 | George Boole | Boolean algebra. Foundation of all digital logic and circuit design. | [I](era-1-mathematical-foundations.md) |
| 1848–1925 | Gottlob Frege | Formal symbolic logic. Foundation of mathematical logic and Turing's work. | [I](era-1-mathematical-foundations.md) |
| 1872–1970 | Bertrand Russell | Principia Mathematica. Logical foundations of mathematics. | [I](era-1-mathematical-foundations.md) |
| 1903–1957 | John von Neumann | Stored-program architecture. All modern computers use the von Neumann design. | [III](era-3-electronic-computer.md) |
| 1903–1995 | Alonzo Church | Lambda calculus. Church-Turing Thesis. Foundation of functional programming. | [II](era-2-theoretical-computer.md) |
| 1906–1992 | Grace Hopper | First compiler. FLOW-MATIC. COBOL. Made programming human-readable. | [III](era-3-electronic-computer.md) |
| 1912–1954 | Alan Turing | Turing machine. Halting problem. Enigma codebreaking. Turing Test. Father of computer science. | [II](era-2-theoretical-computer.md) |
| 1916–2001 | Claude Shannon | Boolean logic → circuits. Information theory. The bit. Foundation of all digital communications. | [II](era-2-theoretical-computer.md) |
| 1924–2007 | John Backus | FORTRAN — first widely used high-level language. BNF grammar notation. | [III](era-3-electronic-computer.md) |
| 1930–2002 | Edsger Dijkstra | Shortest-path algorithm. Structured programming. Semaphores. Dining philosophers. | [IV](era-4-software-revolution.md) |
| 1938– | Donald Knuth | The Art of Computer Programming. TeX. Algorithm analysis as a science. | [IV](era-4-software-revolution.md) |
| 1940/1943– | Ken Thompson | Unix. Go programming language. UTF-8 encoding. | [IV](era-4-software-revolution.md) |
| 1941–2011 | Dennis Ritchie | C programming language. Unix. K&R textbook. | [IV](era-4-software-revolution.md) |
| 1947– | Geoffrey Hinton | Backpropagation. Deep learning. AlexNet. "Godfather of AI." | [VII](era-7-deep-learning.md) |
| 1950– | Steve Wozniak | Apple I, Apple II. Colour-graphics personal-computer architecture. | [V](era-5-personal-computer.md) |
| 1955– | Tim Berners-Lee | World Wide Web. HTTP, HTML, URLs. Made the internet navigable by everyone. | [VI](era-6-internet-and-web.md) |
| 1955–2011 | Steve Jobs | Apple Macintosh. GUI and mouse for the mass market. iPhone. | [V](era-5-personal-computer.md) |
| 1955– | Bill Gates | Microsoft BASIC. MS-DOS. Windows. Software licensing as a business model. | [V](era-5-personal-computer.md) |
| 1960– | Yann LeCun | Convolutional neural networks. LeNet. Foundation of modern computer vision. | [VII](era-7-deep-learning.md) |
| 1969– | Linus Torvalds | Linux kernel. Git. Open-source operating system running 90%+ of cloud infrastructure. | [VI](era-6-internet-and-web.md) |
| 1973– | Larry Page & Sergey Brin | Google. PageRank. Link-based search ranking. Dominant internet advertising. | [VI](era-6-internet-and-web.md) |
