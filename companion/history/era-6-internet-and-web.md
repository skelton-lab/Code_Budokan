# Era VI — The Internet and the Web (1990–2005)

The 1990s transformed computing from a personal tool into a global infrastructure. The World Wide Web, created by Tim Berners-Lee in 1989–1991, made the internet accessible to non-technical users and created the foundation for every online service, business, and community that followed. This era also produced Linux, Java, the browser wars, Google, and the open-source movement — and ended with the dot-com bust that cleared the ground for what came next.

## Profiles

### Tim Berners-Lee (1955– ) · British computer scientist, CERN / MIT

Tim Berners-Lee invented the World Wide Web while working at CERN, the European particle physics laboratory. His 1989 proposal "Information Management: A Proposal" described a hypertext system for sharing information between researchers. By 1991, he had implemented the first web server, the first web browser (WorldWideWeb), and the three foundational protocols: HTTP (HyperText Transfer Protocol), HTML (HyperText Markup Language), and URLs (Uniform Resource Locators). Crucially, Berners-Lee and CERN placed the web's technology in the public domain, releasing it royalty-free. This decision — unlike the patent-protected alternatives — made the web truly universal.

**Key contributions**
- Invented the World Wide Web (1989–1991)
- Created HTTP, HTML, and URLs — the three web protocols
- Built the first web server and web browser
- Made web technology royalty-free / public domain
- Founded the World Wide Web Consortium (W3C)
- Recipient of the Turing Award (2016)

**Legacy.** The web is the largest distributed system in human history. Berners-Lee's decision to release it royalty-free was arguably the most economically impactful act of generosity in history — enabling an estimated $10+ trillion in economic activity. He was knighted in 2004.

**Key work:** *Berners-Lee, T. (1989). Information Management: A Proposal. CERN Internal Memo. Also: Berners-Lee, T. (1991). World Wide Web project. alt.hypertext newsgroup post.*

### Linus Torvalds (1969– ) · Finnish-American software engineer

Linus Torvalds announced Linux in August 1991 as a "hobby" operating system for his personal use. He posted the kernel source code to the comp.os.minix newsgroup and invited contributions. The combination of Torvalds's kernel with the GNU system tools developed by Richard Stallman's Free Software Foundation (which had everything except a kernel) produced a complete Unix-like operating system that was freely available and modifiable. Linux became the operating system of the internet: it runs the overwhelming majority of web servers, all Android devices, the top 500 supercomputers, and most cloud computing infrastructure.

**Key contributions**
- Created the Linux kernel (1991)
- Pioneered the open-source development model via internet collaboration
- Maintained Linux as a benevolent dictator through 30+ years
- Created the Git version control system (2005)
- Linux runs ~90% of cloud infrastructure, 100% of top supercomputers, all Android devices

**Legacy.** Linux is the most widely deployed operating system in the world by any measure that includes servers, supercomputers, and mobile devices. Git — Torvalds's version control system, created in two weeks in 2005 — is the standard tool for all software development.

**Key work:** *Torvalds, L. (1991). What would you like to see most in minix? comp.os.minix newsgroup post, 25 August 1991.*

### Larry Page & Sergey Brin · Page (1973– ) / Brin (1973– ) · co-founders of Google

Larry Page and Sergey Brin were PhD students at Stanford when they developed PageRank — an algorithm that ranks web pages by the number and quality of links pointing to them, rather than by keyword frequency. Their 1998 paper "The Anatomy of a Large-Scale Hypertextual Web Search Engine" described the Google system. PageRank solved the search spam problem that had made earlier search engines increasingly unreliable. Google's search quality was immediately and dramatically better than its competitors. Google launched in 1998; by 2003 it was processing several hundred million searches per day. The company's advertising model — AdWords/AdSense — became the dominant business model of the internet era.

**Key contributions**
- Invented the PageRank algorithm — link-based web page ranking
- Built the Google search engine — dramatically superior to existing engines
- Developed the dominant internet advertising model (AdWords/AdSense)
- Google Maps, Gmail, YouTube (acquired 2006), Android (acquired 2005)
- Google Brain (2011) — one of the founding institutions of deep learning

**Legacy.** Google processes approximately 8.5 billion searches per day. The company's subsequent expansion into cloud computing, AI research (Google Brain, DeepMind), and consumer hardware makes it one of the most influential institutions in the history of the technology industry.

**Key work:** *Brin, S., & Page, L. (1998). The Anatomy of a Large-Scale Hypertextual Web Search Engine. Proceedings of the 7th International Conference on World Wide Web.*

## Milestones

| Year | Event / invention | Significance |
|---|---|---|
| 1989 | World Wide Web proposed — Berners-Lee | HTTP, HTML, URLs conceived. Posted as a proposal to CERN management. |
| 1991 | Linux kernel / WWW goes public | Torvalds releases the Linux kernel. First public web server at CERN. |
| 1993 | Mosaic web browser | NCSA Mosaic. First graphical web browser. Made the web accessible to non-technical users. |
| 1994 | Netscape Navigator / Amazon / Yahoo | Browser wars begin. First major internet companies founded. |
| 1995 | Java (Sun Microsystems) | "Write once, run anywhere." Platform-independent language. Dominates enterprise until the 2010s. |
| 1995 | JavaScript created | Brendan Eich, 10 days at Netscape. Client-side web scripting. Now the web's universal language. |
| 1995 | Apache HTTP Server | Open-source web server. Runs ~40% of all websites at peak. Enables the web hosting industry. |
| 1997 | Deep Blue defeats Kasparov | IBM chess computer beats the world champion. First AI to defeat a human world champion at a major game. |
| 1998 | Google founded / PageRank paper | Brin & Page. Link-based search ranking solves web spam. Transforms information retrieval. |
| 1999 | WiFi standardized (802.11b) | Wireless internet. Mobile computing begins. |
| 2001 | Wikipedia / dot-com bust | User-generated encyclopedia. Open-source knowledge infrastructure. |
| 2003 | WordPress / MySpace | Blogging democratized. Early social media. |
| 2004 | Facebook founded | Zuckerberg, Harvard. Social networking at scale. Transforms online identity and advertising. |
| 2005 | YouTube founded / Git created | Video-sharing platform. Torvalds creates Git in 2 weeks for Linux kernel development. |
