# Era V — The Personal Computer (1975–1989)

The personal computer era made computing personal in a way that the mainframe and minicomputer eras never could. Where a mainframe served an institution, a personal computer served an individual. This shift in scale was a shift in kind: it changed who programmed, what they programmed for, and how software was distributed, sold, and thought about. The companies formed in this era — Apple, Microsoft, IBM — would become the most valuable corporations on Earth.

## Profiles

### Steve Wozniak & Steve Jobs · Wozniak (1950– ) / Jobs (1955–2011) · co-founders of Apple Computer

Steve Wozniak designed the Apple I (1976) and Apple II (1977) almost entirely by himself — an engineering achievement of extraordinary elegance, packing a complete computer with colour graphics and audio into a single chip design far ahead of its time. Steve Jobs provided the commercial vision, marketing genius, and relentless product perfectionism that transformed Wozniak's engineering into a consumer revolution. The Apple II ran VisiCalc — the first spreadsheet — and became the computer that convinced businesses that personal computing was real. The Macintosh (1984), based on ideas from Xerox PARC, introduced the graphical user interface and mouse to the mass market.

**Key contributions**
- Apple II (1977) — first mass-market personal computer with colour graphics
- Macintosh (1984) — first mass-market GUI computer with a mouse
- Popularized the GUI paradigm from Xerox PARC
- Created the personal computing consumer market
- Established software applications (VisiCalc, MacWrite, MacPaint) as the PC value proposition

**Legacy.** The graphical user interface that Apple popularized — and that Microsoft subsequently adapted for Windows — is how billions of people interact with computers today. Jobs's product philosophy ("technology and liberal arts at the intersection") would define Apple's design leadership for decades.

**Key work:** *Herzfeld, A. (1984). Folklore.org — original Macintosh team stories (primary source documentation of the Mac's development).*

### Bill Gates & Paul Allen · Gates (1955– ) / Allen (1953–2018) · co-founders of Microsoft

Bill Gates and Paul Allen founded Microsoft in 1975 after writing a BASIC interpreter for the Altair 8800 personal computer. The critical inflection point came in 1980 when IBM contracted Microsoft to provide the operating system for the IBM PC. Microsoft acquired QDOS (Quick and Dirty Operating System) and licensed it to IBM as PC-DOS — while retaining the right to license it to other manufacturers as MS-DOS. When IBM-compatible clones flooded the market, Microsoft owned the software that ran them all. Windows 1.0 (1985), Microsoft Office, and the subsequent dominance of Windows established Microsoft as the dominant software company of the twentieth century.

**Key contributions**
- BASIC interpreter for the Altair 8800 — Microsoft's founding product (1975)
- MS-DOS — operating system for the IBM PC and all compatibles (1981)
- Windows — graphical operating system for IBM-compatible hardware (1985– )
- Microsoft Office — integrated productivity suite (1989)
- Established the software licensing model as a business paradigm
- Funded and popularized the IBM PC compatible standard

**Legacy.** Microsoft's decision to license MS-DOS rather than sell it exclusively to IBM created the software-only company model: the company that owns the software a billion computers run, without manufacturing a single machine. This business model — replicated by Google (Android), Apple (iOS App Store), and countless others — defines platform economics.

**Key work:** *Gates, B. (1976). An Open Letter to Hobbyists. Homebrew Computer Club Newsletter.*

## Milestones

| Year | Event / invention | Significance |
|---|---|---|
| 1975 | Altair 8800 / Microsoft founded | First personal computer kit. Gates & Allen write BASIC. Microsoft incorporated. |
| 1976 | Apple I | Wozniak's single-board computer. First Apple product. |
| 1977 | Apple II / TRS-80 / Commodore PET | Three mass-market personal computers launched the same year. Personal computing arrives. |
| 1979 | VisiCalc — first spreadsheet | The "killer app" that justified buying an Apple II for business. Transforms office computing. |
| 1980 | Microsoft acquires QDOS / IBM PC deal | Microsoft positioned to own the software on all IBM-compatible PCs. |
| 1981 | IBM PC with MS-DOS | IBM PC sets the hardware standard. MS-DOS runs on all IBM-compatibles. Open architecture enables clones. |
| 1983 | C++ — OOP for systems programming | Stroustrup extends C with classes. C++ becomes dominant for performance-critical OOP. |
| 1984 | Macintosh — GUI mass market | First affordable GUI computer with a mouse. Xerox PARC ideas reach consumers. |
| 1984 | GNU Project — Stallman | Free Software Foundation. GPL licence. Foundation of the open-source software movement. |
| 1985 | Windows 1.0 | Microsoft's GUI for IBM-compatible PCs. |
| 1987 | HyperCard (Apple) | First hypermedia authoring system. Anticipated the web's link structure. |
| 1989 | Microsoft Office launched | Integrated Word, Excel, PowerPoint. Defines the productivity software suite. |
